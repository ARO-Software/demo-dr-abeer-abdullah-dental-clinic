# عيادة الدكتورة عبير عبد الله — Elevate / Dentamaze

Demo website for **عيادة الدكتورة عبير عبد الله لطب وجراحة وتجميل الفم والأسنان** (Dr. Abeer Abdullah Dental Clinic), Al Jubaiha, Amman, Jordan.

- **Category:** dental clinic
- **pitch_type:** site
- **Build date:** 2026-07-16
- **Demo URL:** https://aro-software.github.io/demo-dr-abeer-abdullah-dental-clinic/

## About
Single-page, static, RTL Arabic-first demo. Teal/turquoise medical-aesthetic art direction with a boutique gold accent, typography-forward hero, and one privacy-screened signage photo framed in the credentials block. Mobile-first, animated (IntersectionObserver reveals, sticky condensing header), reduced-motion aware.

Primary CTA books an appointment via WhatsApp: `https://wa.me/962796982720`.

## Stack
Plain HTML + CSS + JS, no backend. Fonts: Tajawal, IBM Plex Sans Arabic, Cormorant Garamond (Google Fonts).

## Files
- `index.html` — markup
- `style.css` — styles
- `script.js` — interactions
- `assets/signage.jpg` — clinic signage (only image used)
